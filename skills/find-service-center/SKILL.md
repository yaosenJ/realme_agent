# 查找服务网点 & 预约维修 Skill

## 1. 触发规则
### 应触发场景
- 查询附近服务网点/服务中心
- 预约到店维修、门店维修
- 询问网点地址、电话、营业时间
- 想知道去哪家网点维修

### 禁止触发场景
- 询问维修进度、维修时间
- 提及“绿线”“屏幕问题”
- 反馈手机不开机、故障问题

## 2. 可用 MCP 工具
本技能配有以下两个mcp工具：

1. get_service_centers
   查询城市/区域的服务网点列表

2. create_instore_appointment
   创建到店维修预约

## 3. 执行流程

### 流程1：用户查询网点
1. 判断是否已提供城市
2. 未提供 → 引导用户说城市
3. 已提供 → 调用 get_service_centers
4. 返回网点名称、地址、电话、营业时间

### 流程2：用户预约维修
1. 先调用 get_service_centers 展示网点
2. 收集：姓名、手机号、预约时间、选择的网点ID
3. 调用 create_instore_appointment
4. 返回预约单号与成功信息

## 5. 示例执行

用户：我想查合肥的服务网点
→ 调用 get_service_centers(city="合肥市")
→ 返回并展示网点信息

用户：帮我预约这个网点
→ 收集姓名、电话、时间
→ 调用 create_instore_appointment
→ 返回预约成功
